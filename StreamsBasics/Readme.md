Java 8 Streams Basics (Separate Classes)

Overview :

This program demonstrates Java 8 Stream operations using a Student model class in a separate file.

---

Files :

Student.java → Student model class
StudentStreamExample.java → Main program

---

Features :

- Print all students
- Filter students with marks > 70
- Find top student
- Count total students
- Get top 3 students

---

Concepts Used :

- Java 8 Streams
- Method references
- Comparator
- Clean code structure
