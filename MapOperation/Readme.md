Java 8 Map Operation (Separate Classes)

Overview :

This program demonstrates transforming a list of strings into their lengths using Java 8 Streams.

---

Files :

NameProcessor.java → Utility class
MapOperationExample.java → Main program

---

Features :

- Convert names to lengths using map()
- Remove duplicates using distinct()
- Sort names
- Count names based on condition

---

Concepts Used :

- Java 8 Streams
- map()
- distinct()
- sorted()
- count()
- Method references
